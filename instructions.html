Instructions

In order for this code to work correctly you need to do the following:

  1. Install the pygame module in the IDE you are using or using the following shell command: "pip install pygame"
  2. Run the code from the main.py file. The button.py file only contains the code for the buttons used in the main menu and the __init__.py file is there to ensure the button.py file will be correctly imported by the main.py and neither file will function on their own.
